## Research Question
Is departure flight traffic more concentrated than return flight traffic around major US holidays?

## Files
- `notebooks/tsa_holiday_analysis.ipynb` - Complete analysis (run in Colab)
- `assets/` - Key visualizations from our findings
- `presentation/` - Slides for our presentation

## Data
This analysis uses live TSA data. Run the notebook to automatically scrape the latest data from [TSA.gov](https://www.tsa.gov/travel/passenger-volumes).

## How to Run
1. [Open the notebook in Colab with this link](https://colab.research.google.com/drive/18sMJfRHCSZC4JWS2uNrVeHUVsiQwhJvY?usp=sharing)
2. Run cells sequentially - the notebook will scrape live TSA data
3. See all analysis results and visualizations

## Key Findings
- Departure traffic is 30% more concentrated than return traffic
- Statistical significance: p < 0.05 across all major holidays
- Consistent pattern observed for Thanksgiving, Christmas, and July 4th
